# Aula 1 — Integração Horizontal e Vertical

## Resumo da Aula
Nesta primeira aula da matéria de **Integração Horizontal e Vertical**, tivemos um resumo explicativo sobre os processos de integração horizontal e integração vertical, com foco na aplicação dentro do meio industrial.

## Estudo de Caso
Durante a aula, a empresa **WEG** — cujo principal produto são motores elétricos — foi utilizada como material de estudo para exemplificar os conceitos apresentados.

## Aplicações Práticas
Também foram apresentadas explicações sobre:

- Como aplicar o conteúdo no dia a dia;
- Possíveis aplicações em empresas de outros ramos;
- Suposições práticas de funcionamento desses modelos de integração em diferentes áreas.
